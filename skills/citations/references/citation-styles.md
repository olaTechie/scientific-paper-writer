# Citation Style Formatting Rules

This reference defines the formatting rules for all six supported citation styles.
Each style is illustrated with the same example paper to facilitate comparison.

**Example paper used throughout:**

> Page MJ, McKenzie JE, Bossuyt PM, Boutron I, Hoffmann TC, Mulrow CD, Shamseer L,
> Tetzlaff JM, Akl EA, Brennan SE, Chou R, Glanville J, Grimshaw JM, Hrobjartsson A,
> Lalu MM, Li T, Loder EW, Mayo-Wilson E, McDonald S, McGuinness LA, Stewart LA,
> Thomas J, Tricco AC, Welch VA, Whiting P, Moher D. The PRISMA 2020 statement:
> an updated guideline for reporting systematic reviews. BMJ. 2021;372:n71.
> doi:10.1136/bmj.n71 PMID: 33782057

---

## 1. Vancouver (default)

**Used by:** Most biomedical journals (JAMA, BMJ, Lancet, Annals of Internal Medicine)

### Format

```
Author AA, Author BB, Author CC. Title. *J Abbrev*. Year;Vol(Issue):Pages. doi:XX PMID: XXXXXXXX
```

### Rules

- **Authors:** List author last name followed by initials without periods or spaces between initials
- **Author count:** If 6 or fewer authors, list all. If more than 6, list the first 6 followed by "et al."
- **Journal name:** Abbreviated per NLM (National Library of Medicine) standard abbreviations. Search the NLM Catalog for the correct abbreviation.
- **Title:** Sentence case (capitalise only the first word and proper nouns)
- **Volume/Issue/Pages:** Format as `Vol(Issue):Pages` — e.g., `372:n71` or `39(23):3120-3130`
- **Page ranges:** Use en-dash (–), not hyphen (-)
- **DOI:** Format as `doi:10.XXXX/XXXXX` — do NOT include the `https://doi.org/` prefix
- **PMID:** Include `PMID: XXXXXXXX` for all PubMed-indexed articles
- **In-text:** Numbered in order of first appearance: [1], [2], [1,3], [4-7]

### Example

```
1. Page MJ, McKenzie JE, Bossuyt PM, Boutron I, Hoffmann TC, Mulrow CD, et al.
   The PRISMA 2020 statement: an updated guideline for reporting systematic reviews.
   *BMJ*. 2021;372:n71. doi:10.1136/bmj.n71 PMID: 33782057
```

## 2. APA (American Psychological Association, 7th edition)

**Used by:** Psychology, social sciences, nursing, education journals

### Format

```
Author, A. A., & Author, B. B. (Year). Title. *Journal*, *Vol*(Issue), Pages. https://doi.org/XX
```

### Rules

- **Authors:** Last name, followed by comma, then initials each followed by a period. Use `&` before the last author.
- **Author count:** If 1-20 authors, list all. If 21+, list the first 19, then an ellipsis (...), then the last author.
- **Journal name:** Full name (not abbreviated), italicised
- **Volume:** Italicised
- **Issue:** In parentheses, not italicised
- **Title:** Sentence case
- **DOI:** Full URL format: `https://doi.org/10.XXXX/XXXXX`
- **Hanging indent:** Second and subsequent lines indented 0.5 inches
- **In-text:** Author-date format: (Page et al., 2021) or Page et al. (2021)

### Example

```
Page, M. J., McKenzie, J. E., Bossuyt, P. M., Boutron, I., Hoffmann, T. C.,
    Mulrow, C. D., Shamseer, L., Tetzlaff, J. M., Akl, E. A., Brennan, S. E.,
    Chou, R., Glanville, J., Grimshaw, J. M., Hrobjartsson, A., Lalu, M. M.,
    Li, T., Loder, E. W., Mayo-Wilson, E., McDonald, S., ... Moher, D. (2021).
    The PRISMA 2020 statement: An updated guideline for reporting systematic
    reviews. *BMJ*, *372*, n71. https://doi.org/10.1136/bmj.n71
```

## 3. AMA (American Medical Association, 11th edition)

**Used by:** JAMA family journals, Archives journals

### Format

```
Author AA, Author BB. Title. *Journal Name*. Year;Vol(Issue):Pages. doi:XX
```

### Rules

- **Authors:** Last name followed by initials without periods. No spaces between initials.
- **Author count:** If 6 or fewer, list all. If more than 6, list the first 3 followed by "et al."
- **Journal name:** Full, unabbreviated journal name, italicised
- **Title:** Sentence case
- **Volume/Issue/Pages:** Same format as Vancouver: `Vol(Issue):Pages`
- **Page ranges:** En-dash (–)
- **DOI:** Format as `doi:10.XXXX/XXXXX` (no URL prefix)
- **PMID:** Not required but may be included
- **In-text:** Superscript numbers in order of first appearance: text^1^ or text^1,3^ or text^4-7^

### Example

```
1. Page MJ, McKenzie JE, Bossuyt PM, et al. The PRISMA 2020 statement: an updated
   guideline for reporting systematic reviews. *The BMJ*. 2021;372:n71.
   doi:10.1136/bmj.n71
```

## 4. Nature

**Used by:** Nature, Nature Medicine, Nature Reviews, and other Nature family journals

### Format

```
Author, A. A. et al. Title. *J. Abbrev.* **Vol**, Pages (Year). https://doi.org/XX
```

### Rules

- **Authors:** Last name, comma, initials with periods. If more than 5 authors, list the first author followed by "et al."
- **Author count:** If 5 or fewer, list all separated by commas with `&` before the last. If more than 5, first author + "et al."
- **Journal name:** Abbreviated (Nature standard abbreviations, similar to NLM), italicised
- **Volume:** Bold
- **Pages:** Not bold, not italicised
- **Year:** In parentheses at the end
- **Title:** Sentence case
- **DOI:** Full URL format: `https://doi.org/10.XXXX/XXXXX`
- **In-text:** Superscript numbers: text^1^ or text^1,3^ or text^4-7^

### Example

```
1. Page, M. J. et al. The PRISMA 2020 statement: an updated guideline for reporting
   systematic reviews. *BMJ* **372**, n71 (2021). https://doi.org/10.1136/bmj.n71
```

## 5. Chicago (Author-Date, 17th edition)

**Used by:** Humanities-adjacent health sciences, public health policy, health economics

### Format

```
Author, FirstName MiddleInitial., and FirstName MiddleInitial. Author. Year. "Title." *Journal* Vol (Issue): Pages. https://doi.org/XX.
```

### Rules

- **Authors:** First author: Last, First Middle. Subsequent authors: First Middle Last. Use "and" before the last author.
- **Author count:** If 1-3 authors, list all. If 4-10, list all in the reference list but use "et al." in-text after the first author. If 11+, list the first 7 followed by "et al." in the reference list.
- **Journal name:** Full name (not abbreviated), italicised
- **Title:** Title case, enclosed in quotation marks
- **Volume:** Not italicised
- **Year:** Immediately after author names
- **DOI:** Full URL format with trailing period
- **In-text:** Author-date in parentheses: (Page et al. 2021) or Page et al. (2021)

### Example

```
Page, Matthew J., Joanne E. McKenzie, Patrick M. Bossuyt, Isabelle Boutron,
    Tammy C. Hoffmann, Cynthia D. Mulrow, Larissa Shamseer, et al. 2021.
    "The PRISMA 2020 Statement: An Updated Guideline for Reporting Systematic
    Reviews." *BMJ* 372: n71. https://doi.org/10.1136/bmj.n71.
```

## 6. IEEE

**Used by:** Biomedical engineering, health informatics, computational biology

### Format

```
[N] A. A. Author, "Title," *J. Abbrev.*, vol. X, no. Y, pp. XX–XX, Month Year. doi:XX
```

### Rules

- **Authors:** Initials first (with periods), then last name. Use "and" before the last author.
- **Author count:** If 6 or fewer, list all. If more than 6, list the first author followed by "*et al.*"
- **Journal name:** Abbreviated (IEEE standard), italicised
- **Title:** Sentence case, enclosed in quotation marks
- **Volume:** Preceded by "vol."
- **Issue:** Preceded by "no."
- **Pages:** Preceded by "pp." with en-dash (–) for ranges
- **Date:** Month (abbreviated) and year
- **DOI:** Format as `doi:10.XXXX/XXXXX`
- **Bracketed number:** [N] at the start, corresponding to in-text citation
- **In-text:** Numbers in square brackets: [1], [2], [1], [3], [4]–[7]

### Example

```
[1] M. J. Page, J. E. McKenzie, P. M. Bossuyt, I. Boutron, T. C. Hoffmann,
    C. D. Mulrow, et al., "The PRISMA 2020 statement: an updated guideline for
    reporting systematic reviews," *BMJ*, vol. 372, p. n71, Mar. 2021.
    doi:10.1136/bmj.n71
```

## Quick Comparison Table

| Feature | Vancouver | APA | AMA | Nature | Chicago | IEEE |
|---|---|---|---|---|---|---|
| In-text format | [1] | (Author, Year) | Superscript^1^ | Superscript^1^ | (Author Year) | [1] |
| Author limit | 6 + et al. | 20 + ... + last | 3 + et al. | 1 + et al. (>5) | 7 + et al. (>10) | 6 + et al. |
| Journal name | NLM abbrev. | Full | Full | Abbrev. | Full | IEEE abbrev. |
| DOI format | doi:10.XX | https://doi.org/ | doi:10.XX | https://doi.org/ | https://doi.org/ | doi:10.XX |
| Volume style | Plain | *Italic* | Plain | **Bold** | Plain | vol. X |
| Title case | Sentence | Sentence | Sentence | Sentence | Title Case | Sentence |
| Title quotes | No | No | No | No | Yes | Yes |
| PMID included | Yes | No | Optional | No | No | No |

## Style Selection Logic

1. Check `PAPER_CONTEXT.md` for `citation_style` field
2. If not specified, check target journal:
   - JAMA, BMJ, Lancet, Annals → Vancouver
   - Nature family → Nature
   - APA journals → APA
   - IEEE journals → IEEE
3. If no journal specified → Vancouver (default)
4. User may override at any time: "format references in APA style"
