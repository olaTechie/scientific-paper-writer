# PRISMA-NMA Extension Checklist

Reference: Hutton B, Salanti G, Caldwell DM, et al. The PRISMA extension
statement for reporting of systematic reviews incorporating network
meta-analyses of health care interventions: checklist and explanations.
*Ann Intern Med*. 2015;162(11):777-784.

This checklist supplements the standard PRISMA 2020 checklist with items
specific to network meta-analyses. Items marked with an asterisk (*) are
new or substantially modified from the original PRISMA checklist.

## Usage Notes

When completing this checklist for a manuscript:
- Replace `[ ]` with `[x]` for items that are reported
- Flag items not reported with a note explaining why (e.g., not applicable)
- Items marked with * are NMA-specific extensions; all others follow standard PRISMA 2020
- This checklist should be included as a supplementary table in the manuscript submission
