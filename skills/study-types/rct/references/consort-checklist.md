## RCT Reporting Checklist (Detailed)

### CONSORT Flow Diagram

Required elements at each stage:

| Stage | Element | Status |
|-------|---------|--------|
| Enrolment | Assessed for eligibility (N) | |
| Enrolment | Excluded (N total), with sub-reasons: not meeting inclusion criteria (N), declined to participate (N), other reasons (N) | |
| Allocation | Randomised (N) | |
| Allocation | Allocated to intervention (N); received allocated intervention (N); did not receive allocated intervention (N, with reasons) | |
| Allocation | Allocated to control (N); received allocated control (N); did not receive allocated control (N, with reasons) | |
| Follow-up | Lost to follow-up (N, with reasons) -- per group | |
| Follow-up | Discontinued intervention (N, with reasons) -- per group | |
| Analysis | Analysed ITT (N) -- per group | |
| Analysis | Excluded from ITT analysis (N, with reasons) -- per group | |
| Analysis | Analysed per-protocol (N) -- per group | |
| Analysis | Excluded from per-protocol analysis (N, with reasons) -- per group | |

### Baseline Characteristics Table

Required elements:

- [ ] Reported by group (intervention vs control), NOT overall only
- [ ] Age: mean +/- SD or median [IQR]
- [ ] Sex: N (%) per category
- [ ] Race/ethnicity: N (%) per category (if relevant and collected)
- [ ] Key clinical variables relevant to the condition
- [ ] Key prognostic factors
- [ ] No p-values for baseline comparisons (balance is by design; use SMD if needed)

### Primary Outcome Reporting

Required elements:

- [ ] Number of events and total per group: events/N (%)
- [ ] Effect estimate (RR, OR, HR, MD, SMD) with 95% CI and p-value
- [ ] Absolute risk in each group
- [ ] Absolute risk difference (ARD) with 95% CI
- [ ] Number needed to treat (NNT) = 1/ARD with 95% CI (where clinically meaningful)
- [ ] For time-to-event: Kaplan-Meier curves referenced, log-rank p-value, median follow-up
- [ ] Analysis population stated (ITT, modified ITT, per-protocol)
- [ ] Forest plot or figure referenced

### Secondary Outcomes

- [ ] Each secondary outcome reported with the same elements as primary
- [ ] Multiplicity adjustment stated (or explicitly stated as not applied)
- [ ] Hierarchy of testing stated (if applicable)

### Adverse Events

- [ ] Any adverse event: N (%) per group
- [ ] Serious adverse events: N (%) per group, with listing
- [ ] Adverse events leading to discontinuation: N (%) per group
- [ ] Deaths: N (%) per group, with causes
- [ ] Most common adverse events (>= X% threshold stated)
- [ ] Safety table referenced
